Documentação do Projeto Tristram Health & Mind

---

 1. Descrição do Projeto

O projeto Tristram Health & Mind é um site institucional com foco na
apresentação e venda de programas voltados ao bem-estar físico e 
mental, incluindo yoga, meditação e flexibilidade.

O sistema permite que o usuário:

• Visualize planos disponíveis
• Acesse detalhes de cada programa
• Simule a compra de um plano
• Escolha formas de pagamento (PIX, boleto ou cartão)
• Receba confirmação de compra

---

 2. Tecnologias Utilizadas

 Front-end
• HTML5
  Utilizado para estruturar todas as páginas do site de forma semântica.

• CSS3
  Responsável pela estilização, layout, responsividade e organização visual.

 Bibliotecas externas

 Font Awesome (CDN)
  Utilizado para ícones (redes sociais, pagamento, etc.).

 6. Como Executar/Testar o Projeto

 Método 1 (mais simples)

1. Baixar ou extrair o projeto
2. Abrir o arquivo `index.html`
3. Navegar normalmente pelo site

 Método 2 (recomendado)

1. Abrir o projeto no Visual Studio Code
2. Instalar a extensão Live Server
3. Clicar com botão direito no `index.html`
4. Selecionar "Open with Live Server"

---

 7. Fluxo de Teste

Para testar completamente o sistema:

1. Acessar a página inicial
2. Escolher um programa
3. Clicar em "Garantir meu acesso agora"
4. Selecionar forma de pagamento:
   • PIX → visualizar QR Code
   • Boleto → baixar PDF
   • Cartão → preencher formulário
5. Finalizar pagamento
6. Verificar página de sucesso
